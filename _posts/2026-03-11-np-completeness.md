We've had (separate) meetings before on both HTMX and FastAPI.  
This time we'll go through using them together in an actual 
(though simple) project.

HTMX is a web technology that makes available an enhanced set of 
HTML5-like elements - call them "HTML5 plus plus".  You can use 
them to write webpages with a lot of the slick dynamic content 
that you would ordinarily use JavaScript for.

Any language can be used for the server side (backend) of an 
HTMX webpage.  We'll use FastAPI, one of those *all the cool 
kids are using it* web frameworks from the Python world.  It 
leverages async programming and Pydantic schema specifications 
to their maximal extent (although that might not come through 
in the simple app we set up).
